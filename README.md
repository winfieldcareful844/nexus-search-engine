## ⚙ Features Explored

- **BM25 Ranking**  
  This method ranks search results by relevance, putting the best matches first.

- **HNSW Vector Search**  
  It allows searching by similarity, useful for finding related items in large data collections.

- **LSM Storage**  
  Stores data efficiently so the tool runs fast but uses less disk space.

- **Bloom Filters**  
  Quickly checks if an item might be present in your data without scanning every file.

- **W-TinyLFU Caching**  
  Speeds up repeated searches by remembering recent results.
